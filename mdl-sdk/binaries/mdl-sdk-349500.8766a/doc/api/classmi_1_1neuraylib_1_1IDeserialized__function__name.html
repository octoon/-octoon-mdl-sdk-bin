virtual const IType_list* mi::neuraylib::IDeserialized_function_name::get_argument_types ( ) const
pure virtual

Returns the argument types of the serialized function call or material instance.

The argument types are identical to the parameter types of the corresponding definition, unless it is one of the Template-like function definitions. The remarks about the expression list for creating calls to the Cast operator apply correspondingly.
