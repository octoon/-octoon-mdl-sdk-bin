pure virtual

Compares two archives for compatibility.

Parameters
[in]archive_fname1Path to first archive.
[in]archive_fname2Path to second archive.
[in]search_pathsAn optional array of additional search paths to consider for archive_fname2.
[out]contextAn execution context. If non-null, the context can be queried for messages after the comparison operation has finished.
Returns
  • 0: The archives are compatible.
  • -1: Invalid parameters (NULL pointer).
  • -2: An error occurred during archive comparison. Please check the context for details.
